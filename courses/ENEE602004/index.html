About This Course

What You Will Learn

From low-level pointer manipulation and linked lists to object-oriented design and real-time network communication — all grounded in Electrical Engineering applications such as sensor logging, ADC calibration, BOM analysis, and embedded control.

🗂️

Data Structures

Linked lists, arrays, and pointer arithmetic for managing sensor data and embedded system state.

⚡

Algorithms

Linear & binary search, bubble/insertion/selection sort, complexity analysis O(n) vs O(log n).

🏗️

Object-Oriented C++

Encapsulation, inheritance, polymorphism, virtual dispatch, and smart pointers (unique_ptr).

Getting Started

Run the Demo Code

Requires g++ with C++17. Each week is self-contained — clone and compile with make.

Clone & compile

git clone https://github.com/CEUI/ENEE602004.git cd ENEE602004/week09_linked_list make # compile all demos ./demo01 # run demo 1

Manual compilation

g++ -std=c++17 -Wall -Isrc -o demo01 src/01_append_prepend.cpp src/sll_ops.cpp ./demo01
